03 - State 与 Props

State 和 Props 是 React 数据管理的核心

Props 是组件之间传递数据的只读属性，State 是组件内部管理的可变状态。两者共同驱动 React 组件的渲染。

Props — 父传子

父组件通过 props 向子组件传递数据

// 子组件接收 props
function Welcome(props) {
  return <h1>你好, {props.name}!</h1>;
}

// 解构写法（推荐）
function Welcome({ name, age }) {
  return (
    <p>{name} 今年 {age} 岁</p>
  );
}

// 父组件传递 props
function App() {
  return (
    <div>
      <Welcome name="小明" age={18} />
      <Welcome name="小红" age={20} />
    </div>
  );
}

Props 默认值

可以为 props 设置默认值

function Button({ children, color = 'blue', disabled = false }) {
  return (
    <button
      style={{ backgroundColor: color }}
      disabled={disabled}
    >
      {children}
    </button>
  );
}

// 使用
<Button>普通按钮</Button>
<Button color="red">红色按钮</Button>
<Button disabled>禁用按钮</Button>

State — useState

useState 是 React 最重要的 Hook，为组件添加状态

import { useState } from 'react';

function Counter() {
  // 声明状态变量 count，初始值 0
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>当前计数: {count}</p>
      <button onClick={() => setCount(count + 1)}>+1</button>
      <button onClick={() => setCount(count - 1)}>-1</button>
      <button onClick={() => setCount(0)}>重置</button>
    </div>
  );
}
调用 setCount 会触发组件重新渲染，React 自动更新 UI。

State 的更新特性

State 更新是异步且批处理的

function Counter() {
  const [count, setCount] = useState(0);

  // 错误用法 — count 不会变为 3
  function handleWrong() {
    setCount(count + 1);
    setCount(count + 1);
    setCount(count + 1); // 最终结果还是 1
  }

  // 正确用法 — 使用函数式更新
  function handleCorrect() {
    setCount(prev => prev + 1);
    setCount(prev => prev + 1);
    setCount(prev => prev + 1); // 最终结果 3
  }

  return (
    <div>
      <p>计数: {count}</p>
      <button onClick={handleWrong}>错误 +3</button>
      <button onClick={handleCorrect}>正确 +3</button>
    </div>
  );
}

事件处理

React 中的事件绑定方式

function EventExample() {
  const [text, setText] = useState('');

  function handleClick(e) {
    alert('按钮被点击了！');
  }

  function handleChange(e) {
    setText(e.target.value);
  }

  function handleSubmit(e) {
    e.preventDefault();
    alert(`提交: ${text}`);
  }

  return (
    <form onSubmit={handleSubmit}>
      <input value={text} onChange={handleChange} />
      <button onClick={handleClick}>点击</button>
      <button type="submit">提交</button>
    </form>
  );
}

状态提升

多个组件共享状态时，将状态提升到最近的共同父组件

function TemperatureInput({ label, value, onChange }) {
  return (
    <div>
      <label>{label}: </label>
      <input value={value} onChange={onChange} />
    </div>
  );
}

function Calculator() {
  const [temp, setTemp] = useState('');

  return (
    <div>
      <TemperatureInput
        label="摄氏"
        value={temp}
        onChange={e => setTemp(e.target.value)}
      />
      <TemperatureInput
        label="华氏"
        value={temp ? (temp * 9/5 + 32).toFixed(1) : ''}
        onChange={e => setTemp(((e.target.value - 32) * 5/9).toFixed(1))}
      />
    </div>
  );
}

小结

  • Props 是只读的，从父组件传给子组件
  • useState 返回 [值, 更新函数] 的数组
  • State 更新触发重新渲染
  • 连续更新时使用函数式 setState(prev => ...)
  • 状态提升是 React 中共享状态的模式

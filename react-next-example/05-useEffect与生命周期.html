05 - useEffect 与生命周期

useEffect 是 React 的副作用处理 Hook

副作用包括数据获取、订阅、DOM 操作、定时器等。useEffect 让你在函数组件中执行副作用操作。

useEffect 基础

useEffect 接收一个回调函数和一个依赖数组

import { useEffect, useState } from 'react';

function Timer() {
  const [seconds, setSeconds] = useState(0);

  useEffect(() => {
    // 组件挂载后执行
    const id = setInterval(() => {
      setSeconds(prev => prev + 1);
    }, 1000);

    // 清理函数 — 组件卸载时执行
    return () => clearInterval(id);
  }, []); // 空依赖 — 只在挂载/卸载时执行

  return <p>已计时: {seconds} 秒</p>;
}
清理函数是防止内存泄漏的关键！清除定时器、取消订阅、中止请求。

依赖数组的行为

不同依赖数组的行为差异

import { useEffect, useState } from 'react';

function EffectDemo() {
  const [count, setCount] = useState(0);
  const [name, setName] = useState('');

  // 1. 无依赖 — 每次渲染都执行
  useEffect(() => {
    console.log('每次渲染都执行');
  });

  // 2. 空数组 — 只在挂载时执行一次
  useEffect(() => {
    console.log('只在挂载时执行');
  }, []);

  // 3. 有依赖 — 依赖变化时执行
  useEffect(() => {
    console.log(`count 变化了: ${count}`);
    document.title = `计数: ${count}`;
  }, [count]); // 只有 count 变化时执行

  // 4. 多个依赖 — 任一变化都执行
  useEffect(() => {
    console.log(`count 或 name 变化: ${count}, ${name}`);
  }, [count, name]);

  return (
    <div>
      <p>计数: {count}</p>
      <button onClick={() => setCount(c => c + 1)}>+1</button>
      <input value={name} onChange={e => setName(e.target.value)} />
    </div>
  );
}

数据请求

使用 useEffect 获取数据

function UserList() {
  const [users, setUsers] = useState([]);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState(null);

  useEffect(() => {
    let cancelled = false;

    async function fetchUsers() {
      try {
        setLoading(true);
        const res = await fetch(
          'https://jsonplaceholder.typicode.com/users'
        );
        if (!res.ok) throw new Error('请求失败');
        const data = await res.json();
        if (!cancelled) {
          setUsers(data);
          setError(null);
        }
      } catch (err) {
        if (!cancelled) setError(err.message);
      } finally {
        if (!cancelled) setLoading(false);
      }
    }

    fetchUsers();

    return () => { cancelled = true; };
  }, []);

  if (loading) return <p>加载中...</p>;
  if (error) return <p>错误: {error}</p>;

  return (
    <ul>
      {users.map(user => (
        <li key={user.id}>{user.name} ({user.email})</li>
      ))}
    </ul>
  );
}
用 cancelled 标志防止组件卸载后更新状态。

生命周期映射

useEffect 与类组件生命周期对比

// 挂载 — componentDidMount
useEffect(() => {
  // 初始化逻辑
}, []);

// 更新 — componentDidUpdate
useEffect(() => {
  // 依赖更新逻辑
}, [dep1, dep2]);

// 卸载 — componentWillUnmount
useEffect(() => {
  return () => {
    // 清理逻辑
  };
}, []);

// 挂载 + 卸载 组合
useEffect(() => {
  console.log('组件挂载');
  return () => console.log('组件卸载');
}, []);

常见用例

useEffect 的实际应用场景

// 1. 监听窗口大小变化
function WindowSize() {
  const [width, setWidth] = useState(window.innerWidth);

  useEffect(() => {
    const handleResize = () => setWidth(window.innerWidth);
    window.addEventListener('resize', handleResize);
    return () => window.removeEventListener('resize', handleResize);
  }, []);

  return <p>窗口宽度: {width}px</p>;
}

// 2. 同步到 localStorage
function useLocalStorage(key, initial) {
  const [value, setValue] = useState(() => {
    const saved = localStorage.getItem(key);
    return saved ? JSON.parse(saved) : initial;
  });

  useEffect(() => {
    localStorage.setItem(key, JSON.stringify(value));
  }, [key, value]);

  return [value, setValue];
}

小结

  • useEffect(fn, deps) — 在组件渲染后执行副作用
  • 无依赖 → 每次渲染执行 / [] → 仅挂载 / [dep] → dep 变化时执行
  • 返回的清理函数在卸载或重新执行前调用
  • 数据请求时注意竞态条件和内存泄漏
  • 可自定义 Hook 封装副作用逻辑

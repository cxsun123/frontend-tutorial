04 - 条件渲染与列表

动态渲染是 React 的核心能力

React 让你可以像写 JavaScript 一样，用条件语句和循环动态生成 UI。

条件渲染方式

四种常见的条件渲染模式

function UserGreeting({ user }) {
  // 1. if/else 语句（推荐用于复杂逻辑）
  if (!user) {
    return <p>请先登录</p>;
  }
  if (user.banned) {
    return <p>账号已被禁用</p>;
  }

  // 2. 三元运算符（简单条件）
  return (
    <div>
      <h2>{user.name ? `欢迎, ${user.name}` : '欢迎, 游客'}</h2>

      {/* 3. 逻辑与 &&（有则显示） */}
      {user.vip && <span className="badge">VIP 会员</span>}

      {/* 4. 逻辑或 ||（无则默认） */}
      <p>角色: {user.role || '普通用户'}</p>
    </div>
  );
}
⚠️ 注意: 0 和空字符串在 && 中会被渲染，请确保左侧是布尔值。

列表渲染与 key

使用数组的 map() 方法渲染列表

function TodoList() {
  const todos = [
    { id: 1, text: '学习 React', done: true },
    { id: 2, text: '写项目', done: false },
    { id: 3, text: '提交代码', done: false },
  ];

  return (
    <ul>
      {todos.map(todo => (
        <li key={todo.id}>
          <span style={{
            textDecoration: todo.done ? 'line-through' : 'none'
          }}>
            {todo.text}
          </span>
        </li>
      ))}
    </ul>
  );
}

key 的重要性

key 帮助 React 识别列表元素的变更

// ✅ 正确: 使用唯一且稳定的 id
{todos.map(todo => (
  <TodoItem key={todo.id} todo={todo} />
))}

// ❌ 错误: 使用索引（导致性能问题和状态混乱）
{todos.map((todo, index) => (
  <TodoItem key={index} todo={todo} />
))}

// ❌ 错误: 不使用 key
{todos.map(todo => (
  <TodoItem todo={todo} />
))}
key 应该在数组中是唯一的且稳定的。不要用 Math.random() 生成 key。

过滤列表

结合 filter() 和 map() 使用

function FilterableList() {
  const [search, setSearch] = useState('');

  const fruits = ['苹果', '香蕉', '橘子', '葡萄', '西瓜'];

  const filtered = fruits.filter(fruit =>
    fruit.includes(search)
  );

  return (
    <div>
      <input
        value={search}
        onChange={e => setSearch(e.target.value)}
        placeholder="搜索水果..."
      />
      <ul>
        {filtered.length > 0 ? (
          filtered.map((fruit, i) => (
            <li key={i}>{fruit}</li>
          ))
        ) : (
          <li>没有匹配的水果</li>
        )}
      </ul>
    </div>
  );
}

表单处理

受控组件 — 表单状态由 React 控制

function LoginForm() {
  const [form, setForm] = useState({
    username: '',
    password: '',
    remember: false
  });

  function handleChange(e) {
    const { name, value, type, checked } = e.target;
    setForm(prev => ({
      ...prev,
      [name]: type === 'checkbox' ? checked : value
    }));
  }

  function handleSubmit(e) {
    e.preventDefault();
    console.log('登录信息:', form);
  }

  return (
    <form onSubmit={handleSubmit}>
      <input name="username" value={form.username}
        onChange={handleChange} placeholder="用户名" />
      <input name="password" type="password" value={form.password}
        onChange={handleChange} placeholder="密码" />
      <label>
        <input name="remember" type="checkbox"
          checked={form.remember} onChange={handleChange} />
        记住我
      </label>
      <button type="submit">登录</button>
    </form>
  );
}

小结

  • 条件渲染: if/else、三元、&&、||
  • 列表渲染: 用 map() 将数据数组转为 JSX 数组
  • 每个列表项必须有一个稳定唯一的 key
  • 受控组件让表单状态由 React 管理
  • filter() + map() 组合使用实现过滤列表

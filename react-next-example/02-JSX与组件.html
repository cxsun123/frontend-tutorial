02 - JSX 与组件

JSX 是 React 的核心语法

JSX 是 JavaScript 的语法扩展，看起来像 HTML，但实际上是 JavaScript 表达式。组件是 React 的构建单元。

JSX 语法规则

JSX 必须遵循几条基本规则

  • 单一根元素 — 必须用一个父标签包裹所有子元素（或用 Fragment <></>）
  • 关闭所有标签 — 即使是 <br> 也要写成 <br />
  • 驼峰命名 — className、onClick、tabIndex
  • 花括号表达式 — 用 {} 嵌入 JavaScript 表达式
function Welcome() {
  const name = "小明";
  return (
    <>
      <h1 className="title">你好, {name}!</h1>
      <p>当前时间: {new Date().toLocaleString()}</p>
      <img src="/avatar.png" alt="头像" />
    </>
  );
}

内联样式

JSX 中使用 JavaScript 对象表示样式

function StyledBox() {
  const boxStyle = {
    backgroundColor: '#f0f0f0',
    padding: '20px',
    borderRadius: '8px',
    color: '#333'
  };

  return (
    <div style={boxStyle}>
      <p>这是一个带内联样式的盒子</p>
    </div>
  );
}

函数组件

React 组件就是一个返回 JSX 的函数

// 定义组件（首字母必须大写）
function Greeting() {
  return <h2>欢迎来到 React!</h2>;
}

function UserCard() {
  return (
    <div className="card">
      <Greeting />
      <p>用户名: 管理员</p>
    </div>
  );
}

// 使用组件
function App() {
  return (
    <div>
      <UserCard />
      <UserCard />
    </div>
  );
}

组件组合

组件可以嵌套组合使用，构建复杂 UI

function Avatar() {
  return <img className="avatar" src="/user.png" alt="用户" />;
}

function UserInfo() {
  return (
    <div>
      <Avatar />
      <h3>用户名</h3>
    </div>
  );
}

function Comment() {
  return (
    <div className="comment">
      <UserInfo />
      <p>这是一条评论内容</p>
    </div>
  );
}

条件渲染

在 JSX 中使用 JavaScript 条件表达式

function StatusMessage({ isLoggedIn }) {
  return (
    <div>
      {isLoggedIn ? (
        <h2>欢迎回来！</h2>
      ) : (
        <button>登录</button>
      )}
    </div>
  );
}

function Notice({ hasNew }) {
  return (
    <div>
      {hasNew && <span>有新消息！</span>}
    </div>
  );
}

小结

  • JSX 在 JS 中写类似 HTML 的标签
  • 用 {} 嵌入任何 JavaScript 表达式
  • 函数组件是接受 props 返回 JSX 的函数
  • 组件首字母必须大写以区分原生标签
  • 组件可以任意组合和复用

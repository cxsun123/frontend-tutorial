01 - React 入门与环境搭建

什么是 React？

React 是 Facebook（Meta）开发的用于构建用户界面的 JavaScript 库。它采用组件化开发模式，使用声明式语法，通过 Virtual DOM 实现高效渲染。React 是目前最流行的前端框架之一。

核心特性

React 的几个关键概念

  • 声明式 — 描述 UI 应该是什么样子，React 负责更新 DOM
  • 组件化 — 将 UI 拆分成独立、可复用的组件
  • Virtual DOM — 通过内存中的虚拟 DOM 树最小化实际 DOM 操作
  • 单向数据流 — 数据从父组件流向子组件
  • JSX — JavaScript 的语法扩展，在 JS 中写 HTML 标签

创建项目

React 官方推荐使用 Vite 或 Create React App 创建项目

# 使用 Vite（推荐，更快）
npm create vite@latest my-app -- --template react

# 进入项目目录
cd my-app

# 安装依赖
npm install

# 启动开发服务器
npm run dev
Vite 是目前最流行的 React 项目脚手架，启动速度快，HMR 热更新体验好。

项目结构

Vite + React 项目的目录结构

my-app/
├── index.html          # 入口 HTML
├── package.json        # 项目配置与依赖
├── vite.config.js      # Vite 配置文件
├── public/             # 静态资源
│   └── vite.svg
└── src/                # 源代码
    ├── main.jsx        # 应用入口 JS
    ├── App.jsx         # 根组件
    └── App.css         # 根组件样式

第一个 React 组件

最简单 React 组件是一个返回 JSX 的函数

// src/App.jsx
function App() {
  return (
    <div>
      <h1>Hello, React!</h1>
      <p>这是我的第一个 React 应用</p>
    </div>
  );
}

export default App;

入口文件

ReactDOM 将根组件渲染到 DOM

// src/main.jsx
import React from 'react';
import ReactDOM from 'react-dom/client';
import App from './App.jsx';

ReactDOM.createRoot(
  document.getElementById('root')
).render(
  <React.StrictMode>
    <App />
  </React.StrictMode>
);

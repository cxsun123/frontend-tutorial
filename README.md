# frontend-tutorial

前端教学示例中心，包含 HTML、CSS、JavaScript 三门课程的示例页面与课件。

## 目录结构

| 目录/文件 | 说明 |
|---|---|
| `index.html` | 课程导航首页 |
| `html-examples/` | HTML 教学示例 |
| `css-examples/` | CSS 教学示例 |
| `js-examples/` | JavaScript 教学示例 |
| `HTML.pptx` | HTML 课程 PPT 课件 |
| `CSS2_CSS3.pptx` | CSS 课程 PPT 课件 |
| `JavaScript.pptx` | JavaScript 课程 PPT 课件 |

## 使用方式

直接在浏览器中打开 `index.html` 即可浏览所有课程内容。

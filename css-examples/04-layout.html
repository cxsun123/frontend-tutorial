04 - 布局定位

理解 CSS 布局的四种核心机制：正常流、display 属性、float 浮动和 position 定位。

1. 正常流 (Normal Flow)

正常流是浏览器默认的布局方式。块级元素从上到下堆叠，行内元素从左到右排列。所有布局技术都是在正常流基础上进行调整。

1.1 块级元素与行内元素

块级元素 1 — 独占一行，宽度撑满父容器
块级元素 2 — 自动换行到下一行
行内元素 A 行内元素 B 行内元素 C ← 行内元素在同一行内排列 
<style>
.flow-demo { display: flex; flex-direction: column; gap: 8px; }
.flow-demo .block {
  background: #eaf2f8; border: 2px solid #3498db;
  padding: 12px; border-radius: 6px;
}
.flow-demo .inline-span {
  background: #e8f8f5; border: 2px solid #27ae60;
  padding: 4px 8px; border-radius: 4px; display: inline;
}
</style>

<div class="flow-demo">
  <div class="block">块级元素 1</div>
  <div class="block">块级元素 2</div>
  <div>
    <span class="inline-span">行内元素 A</span>

  </div>
</div>

2. display 属性

display 是控制布局最核心的属性。block 独占一行可设宽高；inline 行内排列不能设宽高；inline-block 兼具两者特点。

3. Float 浮动

float 最初用于实现文字环绕图片的效果。浮动元素会脱离正常流，向左或向右移动，直到碰到父容器边缘或其他浮动元素。

3.1 图片文字环绕

Float

这是一段围绕浮动元素排列的文字。当元素设置 float: left 后，它会脱离正常流向左浮动，后续的行内内容会环绕在它周围。这就是经典的"文字环绕图片"效果的实现原理。浮动元素虽然脱离了正常流，但仍然会影响行内内容的排列。

这段文字使用了 clear: both，因此它不会环绕浮动元素，而是从浮动元素下方开始排列。 

<style>
.float-box {
  float: left; width: 120px; height: 80px;
  background: #d4efdf; border: 2px solid #27ae60;
  border-radius: 8px; display: flex; align-items: center;
  justify-content: center; margin: 0 16px 8px 0;
}
.cleared { clear: both; }
</style>

<div class="float-box">Float</div>
<p>这是一段围绕浮动元素排列的文字...</p>
<p class="cleared">这段文字使用了 clear: both...</p>

3.2 清除浮动与 clearfix

浮动元素会脱离正常流，导致父容器高度塌陷。clearfix 是经典的解决方案：在父容器上使用 ::after 伪元素清除浮动。

4. Position 定位

position 属性提供更精确的定位能力。static 是默认值；relative 相对自身偏移；absolute 相对最近定位祖先偏移；fixed 固定在视口；sticky 滚动时在阈值前相对、阈值后固定。

4.1 五种定位对比

static（默认）

元素在正常流中，top/left/right/bottom 不生效

static

relative（相对定位）

相对自身原始位置偏移，原空间保留

原始位置
relative

absolute（绝对定位）

脱离文档流，相对最近定位祖先偏移

正常元素
absolute
<style>
/* relative: 相对自身原始位置偏移 */
.pos-relative { position: relative; top: 20px; left: 20px; }

/* absolute: 脱离文档流，相对最近的定位祖先偏移 */
.parent { position: relative; }
.pos-absolute { position: absolute; top: 10px; right: 10px; }
</style>

<div class="pos-relative">relative</div>
<div class="parent">
  <div class="pos-absolute">absolute</div>
</div>

4.2 fixed 固定定位

position: fixed 让元素固定在视口的指定位置，滚动页面时不会移动。常用于固定导航栏、回到顶部按钮等。

点击下方按钮体验 fixed 效果（按钮会固定在页面右上角）：

  right: 20px;
  z-index: 1000;
  background: #e74c3c;
  color: white;
  padding: 12px 20px;
  border-radius: 8px;
}
</style>

<div class="fixed-box">固定在视口右上角</div>

4.3 sticky 粘性定位

position: sticky 是 relative 和 fixed 的混合体。元素在滚动到指定阈值前保持相对定位，到达阈值后变为固定定位。

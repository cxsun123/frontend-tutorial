07 - 过渡与动画

CSS Transition 过渡、CSS Transform 2D 变换、CSS Animation 关键帧动画。

1. CSS Transition 过渡

Transition 让属性值在两个状态之间平滑变化。 最常用于 hover 效果。需要触发条件（如 :hover、:focus、class 切换）才会生效。
颜色变化
圆角变化
阴影扩展
位移弹起

↑ 悬停每个色块观察不同属性的过渡效果 ↑

/* ── transition 四个子属性 ── */
.box {
  transition-property: all;       /* 过渡哪些属性 */
  transition-duration: 0.3s;      /* 持续时间 */
  transition-timing-function: ease; /* 速度曲线 */
  transition-delay: 0s;           /* 延迟多久开始 */
}

/* ── 简写形式 ── */
.box { transition: all .3s ease 0s; }
/*         属性   时长  曲线  延迟 */

/* ── 常用场景 ── */
.btn {
  transition: background .2s, transform .2s;
}
.btn:hover {
  background: #2ecc71;
  transform: translateY(-3px);
}

2. Transition 速度曲线

transition-timing-function 控制过渡的加速度曲线。 ease 是默认值（慢-快-慢），linear 是匀速。
ease（默认）— 0.2s
linear（匀速）— 0.5s
ease-in（慢入）— 0.8s
ease-out（慢出）— 1.2s
ease-in-out（慢入慢出）— 1.6s

↑ 悬停色块观察不同速度曲线的差异 ↑

/* timing-function 关键字 */
transition-timing-function: ease;        /* 默认：慢→快→慢 */
transition-timing-function: linear;      /* 匀速 */
transition-timing-function: ease-in;     /* 慢启动 */
transition-timing-function: ease-out;    /* 慢结束 */
transition-timing-function: ease-in-out; /* 慢启动 + 慢结束 */

/* 贝塞尔曲线（自定义） */
transition-timing-function: cubic-bezier(0.68, -0.55, 0.27, 1.55);

3. CSS Transform 2D 变换

transform 可以对元素进行旋转、缩放、平移、倾斜等变换， 不影响文档流（其他元素位置不变）。常配合 transition 实现平滑动画。

↑ 悬停蓝色方块观察各种变换效果 ↑

/* ── 2D Transform 常用函数 ── */
.box {
  transform: rotate(45deg);        /* 旋转 */
  transform: scale(1.5);           /* 缩放（1=原始大小） */
  transform: translateX(50px);     /* 水平平移 */
  transform: translateY(-30px);    /* 垂直平移 */
  transform: skewX(15deg);         /* X 轴倾斜 */
  transform: skewY(10deg);         /* Y 轴倾斜 */
}

/* ── 多个变换组合（空格分隔）── */
.box {
  transform: rotate(45deg) scale(1.2) translateX(20px);
}

/* ── transform-origin 变换原点 ── */
.box {
  transform-origin: center center; /* 默认中心 */
  transform-origin: top left;      /* 左上角 */
  transform-origin: 50% 50%;       /* 等同 center */
}

4. CSS @keyframes 关键帧动画

@keyframes 定义动画的关键帧序列，通过 animation 属性应用到元素上。 与 transition 不同，animation 无需触发条件，可以自动播放、循环、延迟。
颜色循环
弹跳
旋转
脉冲
滑入
淡入上移
/* ── @keyframes 定义动画帧 ── */
@keyframes bounce {
  0%, 100% { transform: translateY(0); }    /* 开始和结束 */
  50%      { transform: translateY(-30px); } /* 中间 */
}

@keyframes colorShift {
  0%   { background: #3498db; }
  25%  { background: #e74c3c; }
  50%  { background: #f39c12; }
  75%  { background: #27ae60; }
  100% { background: #3498db; }
}

/* ── 应用动画 ── */
.box {
  animation-name: bounce;          /* 引用 @keyframes 名称 */
  animation-duration: 0.8s;        /* 单次时长 */
  animation-iteration-count: infinite; /* 无限循环 */
}

5. animation 属性详解

animation 是 8 个子属性的简写。掌握这些属性可以精确控制动画行为。
delay — 延迟 2s 后开始
2 秒后滑入
direction — alternate 交替方向
来回弹跳
fill-mode — forwards 保留结束状态
动画结束后保留位置
play-state — paused / running
悬停暂停 ↑
/* ── animation 八大子属性 ── */
.box {
  animation-name: myAnim;            /* ① @keyframes 名称 */
  animation-duration: 2s;            /* ② 单次时长 */
  animation-delay: 1s;               /* ③ 延迟多久开始 */
  animation-timing-function: ease;   /* ④ 速度曲线 */
  animation-iteration-count: infinite; /* ⑤ 播放次数（数字 / infinite）*/
  animation-direction: alternate;    /* ⑥ 方向 */
                                       /*   normal    正向 → → */
                                       /*   reverse   反向 ← ← */
                                       /*   alternate 正→反→正  */
                                       /*   alternate-reverse 反→正→反 */
  animation-fill-mode: forwards;     /* ⑦ 结束后保留状态 */
                                       /*   none / forwards / backwards / both */
  animation-play-state: running;     /* ⑧ running / paused */
}

/* ── 简写形式（推荐）── */
/* animation: name duration timing-function delay
              iteration-count direction fill-mode play-state; */
.box {
  animation: myAnim 2s ease 1s infinite alternate forwards;
}

/* 最少写两个值：name + duration */
.box { animation: myAnim 2s; }

6. @keyframes 百分比写法

除了 from/to（等同 0%/100%），可以用百分比定义多个中间关键帧， 制作更复杂的动画序列。
多段变色
渐变背景移动
loading 转圈
/* ── 百分比关键帧 ── */
@keyframes colorShift {
  0%   { background: #3498db; }  /* 开始 */
  25%  { background: #e74c3c; }  /* 25% 处 */
  50%  { background: #f39c12; }  /* 50% 处 */
  75%  { background: #27ae60; }  /* 75% 处 */
  100% { background: #3498db; }  /* 结束 */
}

/* ── 简写：0% 和 100% 可省略 ── */
@keyframes fadeIn {
  from { opacity: 0; transform: translateY(20px); }
  to   { opacity: 1; transform: translateY(0); }
}

/* ── 多步动画（如 loading spinner）── */
@keyframes spin {
  to { transform: rotate(360deg); }
}
.spinner {
  width: 40px; height: 40px;
  border: 4px solid #ecf0f1;
  border-top-color: #3498db;
  border-radius: 50%;
  animation: spin .8s linear infinite;
}

8. 动画性能优化

动画卡顿通常是由于触发了浏览器的重排（reflow）。优先使用只触发合成（composite）的属性。

✅ 高性能（仅合成层）

  • transform — 位移、旋转、缩放
  • opacity — 透明度变化

❌ 低性能（触发重排）

  • width / height
  • margin / padding
  • top / left / right / bottom
  • font-size
/* ── 性能对比 ── */

/* ❌ 用 margin 移动（触发重排） */
.box {
  transition: margin-top .3s;
}
.box:hover { margin-top: -10px; }

/* ✅ 用 transform 移动（仅合成） */
.box {
  transition: transform .3s;
}
.box:hover { transform: translateY(-10px); }

/* ❌ 用 width 变化（触发重排） */
.bar { transition: width .3s; }

/* ✅ 用 scaleX 变化（仅合成） */
.bar { transition: transform .3s; }
/* transform: scaleX(0.5) 等效于 width 缩半 */

/* ── will-change 提示浏览器 ── */
.box {
  will-change: transform, opacity;
  /* 提前告知浏览器将要变化的属性，
     使其提前创建合成层 */

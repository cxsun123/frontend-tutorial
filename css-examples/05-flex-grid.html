05 - Flex 与 Grid

Flexbox 用于一维布局（行或列），Grid 用于二维布局（行和列同时控制）。

1. Flexbox 基础

设置 display: flex 后，容器变为弹性容器，子元素变为弹性项目。弹性项目沿主轴（main axis）排列，默认方向为水平（row）。

7. flex — 弹性分配

flex 是 flex-grow + flex-shrink + flex-basis 的简写。
flex-grow：空间富余时的放大比例（默认 0 不放大）
flex-shrink：空间不足时的缩小比例（默认 1）
flex-basis：初始大小（默认 auto）

7.3 flex-shrink — 缩小比例

当容器空间不足时，flex-shrink 控制项目如何缩小。默认值为 1（等比例缩小），设为 0 则不缩小。

容器宽度固定，项目总宽超出容器 — 看谁缩小更多：

shrink:0
不缩小
shrink:1
默认
shrink:2
缩小2倍

.flex-item-red    { background: #fadbd8; border-color: #e74c3c; color: #922b21; }

flex: auto

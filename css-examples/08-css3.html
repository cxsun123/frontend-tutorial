08 - CSS3 增强特性

圆角、阴影、滤镜、变量、伪类、伪元素等 CSS3 核心增强功能。

1. border-radius 圆角

border-radius 为元素添加圆角。可以分别设置四个角，也可以用百分比创建椭圆或圆形。
border-radius:12px
50%
20px 0
不对称
<style>
.box1 { border-radius: 12px; }           /* 四角统一 */
.box2 { border-radius: 50%; }            /* 椭圆/圆 */
.box3 { border-radius: 20px 0; }         /* 左上 20px, 右上 0 */
.box4 { border-radius: 40px 10px 40px 10px; } /* 顺时针: 左上 右上 右下 左下 */
</style>

<div class="box1">border-radius:12px</div>
<div class="box2">50%</div>
<div class="box3">20px 0</div>

2. box-shadow 盒阴影

box-shadow: x y blur spread color。x/y 必需，blur/spread 可选，inset 表示内阴影。
外阴影
彩色阴影
内阴影
多层阴影

5. animation 动画

@keyframes 定义关键帧，animation 属性控制播放。 属性：animation-name / duration / timing-function / delay / iteration-count / direction / fill-mode

5.1 基础动画 — 弹跳球

<style>
@keyframes bounce {
  0%   { top: 0;   background: #e74c3c; }
  50%  { top: 80px; background: #f39c12; border-radius: 50%; }
  100% { top: 0;   background: #3498db; }
}
.ball {
  width: 50px; height: 50px; border-radius: 12px;
  position: relative;
  animation: bounce 1.2s ease-in-out infinite;
}
</style>

<div class="ball"></div>

6. transform 2D 变换

transform 对元素进行二维变换：translate 移动、rotate 旋转、scale 缩放、skew 倾斜。可组合使用。
translate
rotate
scale
skewX
组合
<style>
.t1 { transform: translate(10px, -10px); }   /* 移动 */
.t2 { transform: rotate(15deg); }            /* 旋转 */
.t3 { transform: scale(1.3); }               /* 放大 1.3 倍 */
.t4 { transform: skewX(-15deg); }            /* X 轴倾斜 */
.t5 { transform: rotate(10deg) scale(1.1) translate(5px, 5px); }
</style>

<div class="t2">rotate</div>
<div class="t3">scale</div>

7. filter 滤镜

filter 对元素应用图形滤镜：blur() 模糊、brightness() 亮度、grayscale() 灰度、contrast() 对比度、saturate() 饱和度。
原图
无滤镜
模糊
blur(3px)
亮
brightness(1.5)
灰度
grayscale(1)
高对比
contrast(2)
色相旋转
hue-rotate(90deg) 
<style>
.box { filter: blur(3px); }        /* 模糊 */
.box { filter: brightness(1.5); }  /* 亮度 150% */
.box { filter: grayscale(1); }     /* 灰度 100% */
.box { filter: contrast(2); }      /* 对比度 200% */
.box { filter: hue-rotate(90deg); }/* 色相旋转 90° */
/* 组合使用 */
.box { filter: blur(1px) brightness(1.1) contrast(1.1); }
</style>

9. box-sizing 盒模型

box-sizing: border-box 让 width/height 包含 padding 和 border，避免手动计算。
content-box（默认）
width: 200px + padding 32px + border 6px
实际宽度 = 238px
border-box
width: 200px（已含 padding + border）
实际宽度 = 200px

两者 CSS width 都是 200px，但实际占用宽度不同

<style>
* { box-sizing: border-box; } /* 推荐全局设置 */

.content-box {
  box-sizing: content-box; /* 默认：width 不含 padding/border */
  width: 200px; padding: 16px; border: 3px solid red;
  /* 实际宽度 = 200 + 32 + 6 = 238px */
}
.border-box {
  box-sizing: border-box;
  width: 200px; padding: 16px; border: 3px solid green;
  /* 实际宽度 = 200px（padding 和 border 包含在内） */
}
</style>

11. 伪类选择器

:nth-child(n)、:nth-last-child(n)、:last-child 等伪类选择器可选取特定位置的元素。 n 可以是数字、关键字（odd/even）或公式（2n+1）。
  • 第 1 项
  • 第 2 项（even）
  • 第 3 项（odd）
  • 第 4 项（even）
  • 第 5 项（odd）— :last-child
<style>
.list li { padding: 8px 12px; margin: 4px 0; border-radius: 6px; }
.list li:nth-child(odd)  { background: #d6eaf8; } /* 奇数行 */
.list li:nth-child(even) { background: #d4efdf; } /* 偶数行 */
.list li:last-child { font-weight: bold; border-left: 4px solid #e74c3c; }
</style>

<ul class="list">
  <li>第 1 项</li>
  <li>第 2 项</li>
  <li>第 3 项</li>
  <li>第 4 项</li>
  <li>第 5 项</li>
</ul>

02 - CSS 盒模型

每个元素在页面中都会生成一个矩形盒子。盒模型从内到外依次是：内容区域 content、内边距 padding、边框 border、外边距 margin。

1. 盒模型总览：四层嵌套关系

内容是最内层；padding 包围内容；border 包围 padding；margin 在 border 外侧，用来推开其他元素。W3C CSS2 规范把这四层分别称为 content box、padding box、border box、margin box。
margin
border
padding
content
content: 元素内容 padding: 内容和边框的距离 border: 盒子的边框 margin: 盒子外部距离 
<style>
.box-model {
  margin: 24px;
  border: 8px solid #c95546;
  padding: 24px;
  background: #d8f0df;
}

.box-model .content {
  min-height: 80px;
  display: grid;
  place-items: center;
  background: #dcecff;
  border: 2px solid #3f7fbf;
  font-weight: bold;
}
</style>

<div class="box-model">
  <div class="content">content 内容区域</div>
</div>

2. Content：width / height 默认控制内容区

默认情况下，width 和 height 设置的是 content 内容区域尺寸，不包含 padding、border、margin。
width: 220px
height: 90px 
<style>
.content-size {
  width: 220px;
  height: 90px;
  padding: 20px;
  border: 8px solid #3498db;
  margin: 16px;
  background: #eaf2f8;
}
</style>

<div class="content-size">
  width 和 height 只指定 content 尺寸
</div>

3. Padding：内边距扩大内容周围空间

padding 位于 content 和 border 之间。元素背景会覆盖 content 和 padding 区域，所以增加 padding 会让背景区域变大。
padding: 6px
padding: 28px
<style>
.padding-small,
.padding-large {
  width: 230px;
  border: 3px solid #27ae60;
  background: #e8f8f5;
}

.padding-small { padding: 6px; }
.padding-large { padding: 28px; }
</style>

<div class="padding-small">padding: 6px</div>
<div class="padding-large">padding: 28px</div>

4. Border：边框在 padding 外侧

border 有宽度、样式、颜色。边框宽度会参与盒子的总尺寸计算；如果 border-style 是 none，边框宽度的计算值会是 0。
2px solid
10px dashed
8px double
<style>
.border-demo {
  width: 180px;
  padding: 14px;
  background: #f4ecf7;
}

.border-solid { border: 2px solid #8e44ad; }
.border-dashed { border: 10px dashed #8e44ad; }
.border-double { border: 8px double #8e44ad; }
</style>

<div class="border-demo border-solid">2px solid</div>
<div class="border-demo border-dashed">10px dashed</div>
<div class="border-demo border-double">8px double</div>

5. Margin：外边距在盒子外部且透明

margin 用来控制元素之间的外部距离。margin 不会显示元素自己的背景，规范中说明 margin 背景始终透明。
margin: 8px
margin: 32px
<style>
.margin-stage {
  background: repeating-linear-gradient(
    45deg,
    #f9fbfd 0 10px,
    #eef3f7 10px 20px
  );
}

.margin-box {
  width: 160px;
  padding: 14px;
  border: 4px solid #e67e22;
  background: #fdebd0;
}

.margin-small { margin: 8px; }
.margin-large { margin: 32px; }
</style>

<div class="margin-stage">
  <div class="margin-box margin-small">margin: 8px</div>
  <div class="margin-box margin-large">margin: 32px</div>
</div>

6. content-box 总尺寸计算

默认 box-sizing: content-box。横向占用空间等于 margin-left + border-left + padding-left + width + padding-right + border-right + margin-right。
width: 240px padding-left + padding-right: 24px + 24px = 48px border-left + border-right: 6px + 6px = 12px margin-left + margin-right: 20px + 20px = 40px 总横向占用: 240 + 48 + 12 + 40 = 340px
content-box 示例
<style>
.content-box-calc {
  box-sizing: content-box;
  width: 240px;
  padding: 24px;
  border: 6px solid #2c3e50;
  margin: 20px;
  background: #eaf2f8;
}

/* 总横向占用:
   20 + 6 + 24 + 240 + 24 + 6 + 20 = 340px */
</style>

<div class="content-box-calc">content-box 示例</div>

7. border-box：width 包含 content + padding + border

box-sizing: border-box 会让 width 包含 content、padding、border。它不包含 margin，但更适合组件和布局，因为设置宽度后不容易被 padding 撑破。
content-box
实际边框外宽度 304px
border-box
实际边框外宽度 240px
<style>
.same-width {
  width: 240px;
  padding: 24px;
  border: 8px solid;
}

.content-box {
  box-sizing: content-box;
  border-color: #3498db;
  background: #eaf2f8;
}

.border-box {
  box-sizing: border-box;
  border-color: #27ae60;
  background: #e8f8f5;
}
</style>

<div class="same-width content-box">content-box: 240 + 48 + 16 = 304px</div>
<div class="same-width border-box">border-box: 边框外宽度仍是 240px</div>

8. Margin 折叠：相邻垂直外边距会合并

普通文档流中的块级元素，如果上下垂直 margin 相邻，可能不会相加，而是折叠成一个 margin。水平 margin 不会折叠。
下 margin: 32px
上 margin: 24px
<style>
.collapse-stage {
  background: #f4f9ff;
  padding: 1px 16px;
}

.top-block {
  height: 50px;
  margin-bottom: 32px;
  background: #dcecff;
}

.bottom-block {
  height: 50px;
  margin-top: 24px;
  background: #fdebd0;
}

/* 两个块之间的垂直距离通常是 32px，而不是 32px + 24px */
</style>

<div class="collapse-stage">
  <div class="top-block">下 margin: 32px</div>
  <div class="bottom-block">上 margin: 24px</div>
</div>

9. 四值简写：上、右、下、左

margin、padding、border-width 都支持 1 到 4 个值。四个值的顺序是 top、right、bottom、left，也就是顺时针方向。
padding: 8px 28px 22px 48px
border-width: 6px 12px 3px 18px 
<style>
.four-values {
  width: 260px;
  padding: 8px 28px 22px 48px;      /* 上 右 下 左 */
  border-width: 6px 12px 3px 18px;  /* 上 右 下 左 */
  border-style: solid;
  border-color: #3498db #27ae60 #e67e22 #8e44ad;
  background: #f8fbff;
}
</style>

<div class="four-values">
  四值简写按顺时针方向读取
</div>

10. 背景绘制范围：background-clip

CSS2 中背景会覆盖 content、padding、border 区域，margin 始终透明。CSS3 的 background-clip 可以控制背景裁剪到 border-box、padding-box 或 content-box。
border-box
padding-box
content-box
<style>
.clip-demo {
  width: 180px;
  padding: 22px;
  border: 10px dashed rgba(52, 152, 219, .75);
  background: #dcecff;
}

.clip-border { background-clip: border-box; }
.clip-padding { background-clip: padding-box; }
.clip-content { background-clip: content-box; }
</style>

<div class="clip-demo clip-border">border-box</div>
<div class="clip-demo clip-padding">padding-box</div>
<div class="clip-demo clip-content">content-box</div>

03 - 颜色与背景

CSS 颜色值、文本颜色、背景色、透明度与渐变。

1. CSS 颜色名称

CSS 支持 140+ 预定义颜色名，可直接用英文单词指定颜色。
Tomato Orange DodgerBlue MediumSeaGreen Gray SlateBlue Violet HotPink

纯红

纯蓝

rgb(红, 绿, 蓝)，每个值 0–255。rgb(255,0,0) 红，rgb(0,255,0) 绿，rgb(0,0,255) 蓝。
rgb(255,0,0)
rgb(0,255,0)
rgb(0,0,255)
rgb(255,165,0)
rgb(128,0,128)

4. RGBA 与透明度

rgba(红, 绿, 蓝, 透明度)，透明度 0（全透明）到 1（不透明）。也可用 opacity 属性控制整个元素的透明度。
1.0
0.75
0.5
0.25
0.1

opacity 属性示例：

1.0
0.6
0.3
<style>
.rgba-demo {
  position: relative;
  height: 120px;
  background: #ecf0f1;
  border-radius: 8px;
}
.rgba-demo > div {
  position: absolute;
  top: 10px;
  width: 100px; height: 100px;
  border-radius: 8px;
  display: flex;
  align-items: center;
  justify-content: center;
  color: white;
  font-weight: bold;
}
/* opacity 控制整个元素（含子元素）透明度 */
.opacity-demo {
  width: 80px; height: 80px;
  background: #3498db;
  border-radius: 8px;
  opacity: 0.5;
  color: white;
}
</style>

5. HSL 颜色

hsl(色相, 饱和度%, 亮度%)。色相 0–360（红→黄→绿→青→蓝→品→红），饱和度 0%（灰）–100%（纯色），亮度 0%（黑）–100%（白）。
hsl(0,100%,50%)
红色
hsl(60,100%,50%)
黄色
hsl(120,100%,50%)
绿色
hsl(200,100%,50%)
天蓝
hsl(270,100%,50%)
紫色

同一色相，不同饱和度和亮度：

.shade-row { display: flex; gap: 8px; margin-top: 14px; }

</style>

6. HSLA 透明色

hsla(色相, 饱和度%, 亮度%, 透明度)，在 HSL 基础上增加第四个参数控制透明度。
hsla(0,...,0.9)
hsla(120,...,0.6)
hsla(220,...,0.4)
hsla(280,...,0.2)

}

}
</style>

8. CSS 文本颜色 color

color 属性设置文本（前景）颜色。

9. 线性渐变 linear-gradient

linear-gradient(方向, 颜色1, 颜色2, ...)。方向可以是角度（45deg）或关键字（to right, to bottom right 等）。
linear-gradient(to right, 红, 橙, 绿)
linear-gradient(135deg, #667eea, #764ba2)
linear-gradient(to bottom, 深蓝 → 浅蓝)

radial-gradient(形状, 颜色1, 颜色2, ...)。形状可以是 circle（圆形）或 ellipse（椭圆，默认）。
circle
circle 暖色
ellipse

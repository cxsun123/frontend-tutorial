01 - CSS 选择器与层叠

CSS 选择器用于选取 HTML 元素并应用样式。本章按教学常用分类讲解：简单选择器、组合器选择器、属性选择器、伪类选择器、伪元素选择器，以及分组和特异性。

1. 选择器分类总览

常见分类包括：简单选择器、组合器选择器、属性选择器、伪类选择器、伪元素选择器。学习时先问两个问题：我要选谁？选择条件是什么？
简单：p .class #id * 组合器：空格 > + ~ 属性：[attr] [attr=value] 伪类：:hover :checked :nth-child() 伪元素：::before ::first-line 
<style>
/* 简单选择器 */
p { color: #34495e; }
.important { color: #e67e22; }
#intro { background: #eaf2f8; }

/* 组合器选择器 */
.article p { line-height: 1.8; }
.article > h3 { color: #2980b9; }

/* 属性、伪类、伪元素 */
a[target="_blank"] { font-weight: bold; }
button:hover { background: #2ecc71; }
p::first-letter { font-size: 2em; }
</style>

2. 简单选择器：元素、类、ID、通用

元素选择器选一类标签；类选择器适合复用；ID 选择器用于页面中唯一元素；通用选择器 * 会命中所有元素，通常要谨慎使用。

元素选择器：所有段落

类选择器：可复用提示

ID 选择器：唯一提示

<style>
.simple-scope * {
  box-sizing: border-box;
}

.simple-scope p {
  padding: 8px 10px;
  border-radius: 6px;
  background: #f4f9ff;
}

.simple-scope .notice {
  color: #e67e22;
  font-weight: bold;
}

#unique-tip {
  border-left: 4px solid #3498db;
}
</style>

<div class="simple-scope">
  <p>元素选择器：所有段落</p>
  <p class="notice">类选择器：可复用提示</p>

3. 多类、分组与选择器链

多个类连写表示“同时具备这些类”；逗号分组表示“这些选择器共享同一组样式”。
普通标签 同时有 tag 和 hot 链接

<div class="chain-demo">
  <span class="tag">普通标签</span>

4. 组合器选择器：后代、子代、相邻兄弟、通用兄弟

组合器描述元素之间的结构关系：空格选后代，> 选直接子元素，+ 选紧邻的下一个兄弟，~ 选后续所有兄弟。

章节标题

标题后的第一段

标题后的第二段

嵌套段落

<style>
/* 后代：容器内部所有 p */
.combinator-demo p {
  color: #34495e;
}

/* 子代：只选直接子元素 p，不选嵌套 p */
.combinator-demo > p {
  background: #eaf2f8;
  padding: 8px;
}

/* 相邻兄弟：h3 后面紧挨着的第一个 p */
.combinator-demo h3 + p {
  border-left: 4px solid #3498db;
}

/* 通用兄弟：h3 后面所有同级 p */
.combinator-demo h3 ~ p {
  border-radius: 6px;
}
</style>

<div class="combinator-demo">
  <h3>章节标题</h3>
  <p>标题后的第一段</p>
  <p>标题后的第二段</p>
  <div><p>嵌套段落</p></div>
</div>

5. 属性选择器：按属性和值匹配

属性选择器适合表单、链接、data-* 状态等场景。常见写法包括存在、等于、包含单词、包含片段、开头匹配、结尾匹配。
外部链接 锚点链接 
<style>
/* 存在 target 属性 */
.attr-demo [target] {
  font-weight: bold;
}

/* href 以 https 开头 */
.attr-demo a[href^="https"] {
  color: #27ae60;
}

/* href 以 # 开头 */
.attr-demo a[href^="#"] {
  color: #8e44ad;
}

/* data-state 的空格分隔列表中包含 active */
.attr-demo [data-state~="active"] {
  background: #2ecc71;
}

/* type 等于 email */
.attr-demo input[type="email"] {
  border: 2px solid #3498db;
}
</style>

<div class="attr-demo">
  <a href="https://example.com" target="_blank">外部链接</a>
  <a href="#local">锚点链接</a>
  <button data-state="active primary">激活按钮</button>
  <input type="email" placeholder="email@example.com">
</div>

6. 状态伪类：链接、悬停、焦点、表单状态

伪类以冒号开头，用于描述状态或结构。状态伪类常用于交互反馈，例如 :hover、:focus、:checked、:disabled。

7. 结构伪类：first、last、nth、not、has

结构伪类根据元素在父级中的位置或关系匹配元素。:has() 可根据子元素反向选择父级，适合现代浏览器。
  • 第一项
  • 第二项
  • 第三项 featured
  • 第四项
  • 最后一项
<style>

  font-weight: bold;
}

.structure-demo li:last-child {
  color: #8e44ad;
}

.structure-demo li:nth-child(even) {
  background: #f4f9ff;
}

.structure-demo li:not(.featured) {
  border-left: 3px solid #d5dbdb;
}

.structure-demo:has(.featured) {
  border: 1px dashed #f39c12;

8. 伪元素：before、after、first-letter、selection、marker

伪元素以双冒号开头，用于选择元素的一部分或创建装饰性内容。常用于标记、引号、首字母、选中文本和列表符号。

伪元素标题

选择这段文字，观察 selection 样式。首字母会被放大。

  • marker 可修改列表符号
  • before / after 可生成装饰内容
<style>
.pseudo-demo h3::before {
  content: "§ ";
  color: #3498db;
}

.pseudo-demo h3::after {
  content: " / selector";
  color: #95a5a6;
  font-size: .8em;
}

.pseudo-demo p::first-letter {
  font-size: 2.4em;
  color: #e67e22;
  float: left;
  line-height: 1;
  margin-right: 4px;
}

.pseudo-demo ::selection {
  background: #ffeaa7;
}

.pseudo-demo li::marker {
  color: #27ae60;
}
</style>

<article class="pseudo-demo">
  <h3>伪元素标题</h3>
  <p>选择这段文字，观察 selection 样式。首字母会被放大。</p>
  <ul>
    <li>marker 可修改列表符号</li>
    <li>before / after 可生成装饰内容</li>
  </ul>
</article>

9. 特异性与层叠：谁最终生效

当多个规则命中同一元素时，浏览器按来源、重要性、特异性和出现顺序决定最终样式。大致上：ID 高于 class/属性/伪类，高于元素/伪元素。

观察最终颜色和边框

<style>
p {
  color: #34495e;
}

.message {
  color: #27ae60;
}

.message.warning {
  border-left: 4px solid #f39c12;
}

#priority {
  color: #e74c3c;
}
</style>

<p id="priority" class="message warning">观察最终颜色和边框</p>

+ 只选紧邻的下一个兄弟，~ 选后续所有同级兄弟。* + * 常用于给容器内除第一个元素外的所有元素加间距。

:nth-child() 看元素在所有兄弟中的位置；:nth-of-type() 只看同类型元素的位置。两者看似相近，命中结果可能不同。

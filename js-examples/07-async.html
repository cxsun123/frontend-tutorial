07 - 异步编程

定时器、Promise 与 async/await。

1. async / await

async/await 让异步流程更接近同步写法。
<button onclick="loadData()">加载模拟数据</button>
<pre id="asyncOut"></pre>
<script>
function mockFetch(){
  return new Promise(function(resolve){
    setTimeout(function(){ resolve({ id: 1, title: '异步数据' }); }, 600);
  });
}
async function loadData(){
  const data = await mockFetch();
  document.getElementById('asyncOut').textContent = JSON.stringify(data, null, 2);
}
</script>
2. 数组处理
map 和 filter 常用于处理列表。
<button onclick="runDemo6_1()">处理数组</button>
<pre id="out6_1"></pre>
<script>
function runDemo6_1(){
  const scores = [82,95,67,90];
  const passed = scores.filter(function(score){ return score >= 80; });
  document.getElementById('out6_1').textContent = '优秀成绩：' + passed.join(', ');
}
</script>

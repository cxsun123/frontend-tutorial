05 - DOM操作

查询元素、修改内容、创建节点。

1. 变量与输出

变量保存数据，DOM 用于显示结果。
<button onclick="runDemo4_0()">运行</button>
<div id="out4_0" class="output"></div>
<script>
function runDemo4_0(){
  let count = 1;
  count += 2;
  const name = 'JavaScript';
  document.getElementById('out4_0').textContent = name + ' count = ' + count;
}
</script>

2. 数组处理

map 和 filter 常用于处理列表。
<button onclick="runDemo4_1()">处理数组</button>
<pre id="out4_1"></pre>
<script>
function runDemo4_1(){
  const scores = [82,95,67,90];
  const passed = scores.filter(function(score){ return score >= 80; });
  document.getElementById('out4_1').textContent = '优秀成绩：' + passed.join(', ');
}
</script>
3. 交互反馈
事件触发函数，函数修改页面。
<button onclick="runDemo4_2()">点击反馈</button>
<div id="out4_2" class="output"></div>
<script>
function runDemo4_2(){
  document.getElementById('out4_2').textContent = '事件已触发：' + new Date().toLocaleTimeString();
}
</script>

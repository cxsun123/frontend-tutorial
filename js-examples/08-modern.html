08 - 现代JS

localStorage、JSON 和模块思想。

1. localStorage

localStorage 可以保存字符串数据。
<input id="noteInput" placeholder="输入笔记">
<button onclick="saveNote()">保存</button>
<div id="noteOut" class="output"></div>
<script>
function saveNote(){
  const value = document.getElementById('noteInput').value;
  localStorage.setItem('demo-note', value);
  document.getElementById('noteOut').textContent = '已保存：' + localStorage.getItem('demo-note');
}
</script>

2. 数组处理

map 和 filter 常用于处理列表。
<button onclick="runDemo7_1()">处理数组</button>
<pre id="out7_1"></pre>
<script>
function runDemo7_1(){
  const scores = [82,95,67,90];
  const passed = scores.filter(function(score){ return score >= 80; });
  document.getElementById('out7_1').textContent = '优秀成绩：' + passed.join(', ');
}
</script>

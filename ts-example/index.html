什么是 TypeScript？

TypeScript 是 JavaScript 的超集，添加了静态类型系统。它在编译时检查错误，编译为纯 JavaScript 运行。

npm i -g typescript tsc 01-基本类型.ts node 01-基本类型.js

快速开始

# 安装 TypeScript
npm install -g typescript

# 编译单个文件
tsc src/01-基本类型.ts

# 使用 tsconfig.json 编译全部
tsc

# 运行编译后的 JS
node dist/01-基本类型.js

# 监听模式编译
tsc --watch

04 - 文本格式化元素

HTML 提供了丰富的文本语义元素，用于表达文本的含义和结构。

1. 标题 h1 ~ h6

标题从 <h1> 到 <h6>，级别依次降低。<h1> 最重要，通常每页只有一个。

h1 - 一级标题

h2 - 二级标题

h3 - 三级标题

h4 - 四级标题

h5 - 五级标题
h6 - 六级标题
<h1>一级标题</h1>
<h2>二级标题</h2>
<h3>三级标题</h3>
<h4>四级标题</h4>
<h5>五级标题</h5>
<h6>六级标题</h6>

2. 段落、换行与水平线

这是第一个段落。段落之间会自动产生间距。<p> 标签定义一个段落。

这是第二个段落。浏览器会在每个 <p> 元素前后添加空行。

第一行文本
第二行文本
第三行文本

上面是一条水平分割线 <hr>，用于分隔内容。

<p>段落文本</p>

<p>第一行<br>第二行</p>

<hr>  <!-- 水平分割线 -->

3. 粗体、斜体与强调

<strong> 表示重要性（语义上强调），<b> 仅视觉加粗。<em> 表示重音强调，<i> 仅视觉倾斜。

strong：这段文字很重要（语义强调，浏览器显示为粗体）

b：这段文字被加粗（仅视觉效果，无语义）

em：这段文字需要强调（语义强调，浏览器显示为斜体）

i：这段文字是斜体（仅视觉效果，常用于外来词、术语等）

<strong>重要内容</strong>  <!-- 语义粗体 -->
<b>加粗文字</b>            <!-- 视觉粗体 -->

<em>强调文字</em>          <!-- 语义斜体 -->
<i>斜体文字</i>            <!-- 视觉斜体 -->

4. 标记、代码与预格式化

mark 标签 用于高亮标记文本。

code 标签 用于显示行内代码。

pre 标签
保留文本的
换行和空格

samp 标签 表示程序输出。

Ctrl + S kbd 标签 表示键盘输入。

var 标签 表示数学/编程变量。

<mark>高亮文字</mark>     <!-- 高亮标记 -->
<code>console.log()</code> <!-- 行内代码 -->
<samp>Error 404</samp>    <!-- 程序输出 -->
<kbd>Ctrl+C</kbd>         <!-- 键盘按键 -->
<var>x</var>              <!-- 变量 -->

<pre>
预格式化文本
  保留空格和换行
</pre>

5. 引用

<q> 行内引用（自动加引号），<blockquote> 块级引用，<cite> 引用作品标题。

鲁迅说过：世上本没有路，走的人多了便成了路。

学而不思则罔，思而不学则殆。——这是块级引用，通常有缩进效果。

引用作品：《红楼梦》 是中国古典四大名著之一。

<q>行内引用</q>

<blockquote cite="source">
  块级引用内容
</blockquote>

<cite>《红楼梦》</cite>

6. 上标、下标与其他语义标签

H2O — <sub> 下标

E=mc2 — <sup> 上标

删除线文字 — <del> 删除

下划线新增文字 — <ins> 插入

不再准确的信息 — <s> 失效删除

small 小号文字，常用于版权等辅助信息

HTML — <abbr> 缩写（鼠标悬停显示全称）

2024年元旦 — <time> 机器可读时间

H<sub>2</sub>O           <!-- 下标 -->
E=mc<sup>2</sup>         <!-- 上标 -->
<del>删除文字</del>       <!-- 删除线 -->
<ins>插入文字</ins>       <!-- 下划线 -->
<s>失效文字</s>           <!-- 失效删除 -->
<small>小号文字</small>    <!-- 辅助信息 -->
<abbr title="全称">缩写</abbr> <!-- 缩写 -->
<time datetime="2024-01-01">日期</time>

7. 容器元素：span 和 div

<span> 行内容器，不换行。<div> 块级容器，独占一行。两者无语义，常配合 CSS 使用。

这是 红色加粗的行内文字，前后不会换行。

这是一个 div 块级容器，独占一行，常用于布局分组。 
<span style="color:red">行内文本</span>

<div style="background:#eee">
  块级容器，独占一行
</div>

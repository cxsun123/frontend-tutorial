08 - 事件处理器属性

注意：内联事件处理器（如 onclick）便于教学演示，但实际项目中推荐使用 JavaScript 的 addEventListener。
事件日志：

1. onclick - 点击事件

点击这段文字试试（cursor:pointer 提示可点击） 

<button onclick="handleClick()">点击我</button>

<p onclick="alert('段落被点击')" style="cursor:pointer">
  可点击的段落
</p>

<script>
function handleClick() {
  alert('按钮被点击了');
}
</script>

2. onmouseover / onmouseout - 鼠标悬停

鼠标移入测试 
<div
  style="width:220px; padding:24px; color:white; background:#3498db; border-radius:8px;"
  onmouseover="this.style.background='#e74c3c'; this.textContent='鼠标悬停中'"
  onmouseout="this.style.background='#3498db'; this.textContent='鼠标移入/移出测试'"
>
  鼠标移入/移出测试
</div>

3. oninput - 实时输入

oninput 在输入值每次改变时触发（实时响应），onchange 在失焦或回车时触发。

输入内容：(空)

<input oninput="updateText(this.value)">
<p id="output"></p>

<script>
function updateText(val) {
  document.getElementById('output').textContent = val;
}
</script>

<!-- onchange 在失焦时触发 -->
<input onchange="alert(this.value)">

4. onkeydown / onkeyup - 键盘事件

<input
  onkeydown="logKey(event.key, event.keyCode)"
  onkeyup="logKeyUp(event.key)"
  placeholder="按键测试"
>
<p id="keyOutput">等待用户按下键盘</p>

<script>
function logKey(key, keyCode) {
  document.getElementById('keyOutput').textContent =
    'keydown: ' + key + ' (keyCode: ' + keyCode + ')';
}

function logKeyUp(key) {
  document.getElementById('keyOutput').textContent =
    'keyup: ' + key;
}
</script>

5. onfocus / onblur - 焦点事件

<input
  style="border:2px solid #bdc3c7; padding:8px 12px; border-radius:6px;"
  onfocus="this.style.borderColor='green'"
  onblur="this.style.borderColor='gray'"
  placeholder="焦点测试"
>

6. onsubmit - 表单提交

<form onsubmit="handleSubmit(event)">
  <input name="data">
  <button type="submit">提交</button>
</form>

<script>
function handleSubmit(event) {
  event.preventDefault(); // 阻止默认提交行为
  alert('表单提交已被 JavaScript 拦截');
}
</script>

7. onchange - 下拉选择变化

<select onchange="handleSelect(this.value)">
  <option value="">-- 请选择 --</option>
  <option value="html">HTML</option>
  <option value="css">CSS</option>
  <option value="js">JavaScript</option>
</select>
<span id="selectOutput" style="margin-left:12px;"></span>

<script>
function handleSelect(value) {
  document.getElementById('selectOutput').textContent =
    value ? '你选择了：' + value : '尚未选择';
}
</script>

8. onload - 页面/资源加载完成

onload 在页面完全加载后触发：

页面本身也触发了 onload 事件（见下方日志） 
<p id="loadStatus">等待图片加载完成</p>

<img src="photo.jpg"
  width="200"
  alt="测试图片"
  onload="log('图片加载完成')"
  onerror="log('图片加载失败')"
>

<script>
function log(message) {
  document.getElementById('loadStatus').textContent = message;
}

window.onload = function init() {
  console.log('页面加载完成');
};
</script>

9. 常用事件处理器参考

属性触发时机常用元素
onclick点击元素时所有元素
ondblclick双击元素时所有元素
onmouseover鼠标移入所有元素
onmouseout鼠标移出所有元素
oninput输入值实时变化input, textarea
onchange值改变且失焦input, select, textarea
onfocus获得焦点input, textarea, select
onblur失去焦点input, textarea, select
onkeydown键盘按下input, textarea, document
onkeyup键盘释放input, textarea, document
onsubmit表单提交form
onload资源加载完成img, body, script
onerror加载出错img, script, iframe
onscroll滚动时div, window
onresize窗口/元素大小变化window

05 - 列表与链接元素

列表用于组织信息，链接是互联网的核心——连接各个页面和资源。

1. 无序列表 <ul>

<ul> 定义无序列表，<li> 定义列表项。项目符号由浏览器默认样式决定（通常为圆点）。
  • HTML - 网页结构
  • CSS - 样式表现
  • JavaScript - 交互行为
<ul>
  <li>HTML - 网页结构</li>
  <li>CSS - 样式表现</li>
  <li>JavaScript - 交互行为</li>
</ul>

2. 有序列表 <ol>

<ol> 定义有序列表。type 属性指定编号类型（1/A/a/I/i），start 指定起始值，reversed 倒序。

默认数字编号：

  1. 学习 HTML
  2. 学习 CSS
  3. 学习 JavaScript

大写字母编号（type="A"）：

  1. 第一步
  2. 第二步
  3. 第三步

罗马数字编号（type="I"）：

  1. 第一章
  2. 第二章

从 5 开始（start="5"）：

  1. 第五项
  2. 第六项

倒序列表（reversed）：

  1. 第三
  2. 第二
  3. 第一
<ol>                  <!-- 默认数字 -->
  <li>项目</li>
</ol>

<ol type="A">         <!-- 大写字母 -->
  <li>项目</li>
</ol>

<ol type="I">         <!-- 罗马数字 -->
  <li>项目</li>
</ol>

<ol start="5">        <!-- 从5开始 -->
  <li>项目</li>
</ol>

<ol reversed>         <!-- 倒序 -->
  <li>项目</li>
</ol>

3. 嵌套列表

列表可以嵌套使用，形成层级结构。
  • 前端开发
    • HTML
    • CSS
      • Flexbox
      • Grid
    • JavaScript
  • 后端开发
    • Node.js
    • Python
<ul>
  <li>前端开发
    <ul>
      <li>HTML</li>
      <li>CSS</li>
    </ul>
  </li>
</ul>

4. 定义列表 <dl>

<dl> 定义列表，<dt> 定义术语，<dd> 定义描述。
HTML
超文本标记语言，用于构建网页的结构。
CSS
层叠样式表，用于控制网页的样式和布局。
JavaScript
一种编程语言，为网页添加交互功能。
<dl>
  <dt>HTML</dt>
  <dd>超文本标记语言</dd>

  <dt>CSS</dt>
  <dd>层叠样式表</dd>
</dl>

5. <a> 超链接

href 指定链接地址，target 控制打开方式，download 下载文件，rel 定义关系。
  • MDN Web Docs (新标签页打开)
  • 跳转到页面底部（锚点链接）
  • 发送邮件 (mailto:)
  • 拨打电话 (tel:)
  • 同项目相对链接

图片链接：

<!-- 基础链接 -->
<a href="https://example.com">访问网站</a>

<!-- 新标签页打开 -->
<a href="https://example.com" target="_blank" rel="noopener">
  新标签页打开
</a>

<!-- 锚点链接 -->
<a href="#section-id">跳转到某处</a>
<div id="section-id">目标位置</div>

<!-- 邮件和电话 -->
<a href="mailto:user@example.com">发邮件</a>
<a href="tel:13800000000">打电话</a>

<!-- 图片链接 -->
<a href="https://example.com">
  <img src="photo.jpg" alt="示例图片" width="160">
</a>

6. <nav> 导航列表

  • 首页
  • 表单
  • 表格
  • 多媒体
<nav>
  <ul>
    <li><a href="/">首页</a></li>
    <li><a href="/about">关于</a></li>
  </ul>
</nav>

07 - 全局属性

全局属性可以应用于任何 HTML 元素，用于标识、样式控制和行为配置。

1. id 和 class

id 在页面中唯一标识一个元素，class 可为多个元素设置相同的样式类。

这个段落有 id="unique-para"（页面唯一）

这个段落有 class="highlight-box"（可复用）

另一个同 class 的段落

<p id="unique-para">唯一 ID</p>
<p class="highlight-box">可复用的 class</p>

<!-- CSS 中使用 -->
<style>
  #unique-para { color: blue; }
  .highlight-box { background: yellow; }
</style>

2. style 内联样式

style 属性直接在元素上定义 CSS 样式。优先级最高，但不推荐大量使用。

style="color: red; font-size: 1.1em; ..."

style="border: 2px dashed blue; padding: 10px; ..." 

<p style="color: red; font-size: 1.1em;">内联样式</p>

<!-- 常用 style 属性 -->
<div style="
  color: #333;
  background: #f5f5f5;
  padding: 10px;
  margin: 5px;
  border: 1px solid #ddd;
  border-radius: 4px;
  font-size: 14px;
">样式化的 div</div>

3. title 提示文本

title 属性为元素提供额外信息，通常显示为鼠标悬停时的提示框。

鼠标悬停在此文字上，查看 title 提示（第1行）

HTML（鼠标悬停查看解释）

span 元素（鼠标悬停查看） 
<p title="这是提示信息">鼠标悬停查看</p>
<abbr title="HyperText Markup Language">HTML</abbr>

4. hidden 隐藏元素

hidden 属性使元素不显示在页面上，但仍保留在 DOM 中。

上面这段文字在 HTML 中是隐藏的：

你看不到我！（hidden 属性生效）

这段文字正常显示。

我被隐藏了，点击按钮让我出现！

<p hidden>这个段落不会显示</p>

<button onclick="toggleHidden()">切换显示/隐藏</button>
<p id="hiddenDemo" hidden>我被隐藏了，点击按钮让我出现！</p>

<script>
function toggleHidden() {
  const element = document.getElementById('hiddenDemo');
  element.hidden = !element.hidden;
}
</script>

5. data-* 自定义数据属性

data-* 允许在元素上存储自定义数据，通过 dataset API 访问。
这个元素有自定义数据属性

商品信息

<div id="user" data-user-id="1001" data-role="admin">
  用户信息
</div>

<script>
  const el = document.getElementById('user');
  console.log(el.dataset.userId);   // "1001"
  console.log(el.dataset.role);     // "admin"
</script>

6. tabindex 键盘导航

tabindex 控制 Tab 键焦点顺序。值为 -1 时不参与 Tab 导航，0 按 DOM 顺序，正数按值排序。

用 Tab 键测试焦点顺序：

tabindex=-1（按 Tab 跳过此项） 

<input tabindex="0">      <!-- 默认顺序 -->
<button tabindex="1">第一个</button>
<button tabindex="2">第二个</button>
<p tabindex="-1">不参与Tab</p>

7. lang / dir / draggable

lang - 指定元素内容的语言：

这是中文段落。

This is an English paragraph.

これは日本語の段落です。

dir - 文字方向：

默认从左到右：Hello World

从右到左：مرحبا بالعالم

自动检测：Hello مرحبا

draggable - 可拖拽：

拖拽我！ draggable="true" 
<p lang="en">English text</p>
<p lang="ja">日本語テキスト</p>

<p dir="ltr">从左到右</p>
<p dir="rtl">من اليمين لليسار</p>

<div draggable="true">可拖拽</div>

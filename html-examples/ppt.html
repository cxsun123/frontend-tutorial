一、HTML 基础回顾

HTML = HyperText Markup Language（超文本标记语言）


HTML 文档由元素（Element）组成，每个元素包含：

  • 开始标签：如 <p>
  • 内容：标签之间的文本或子元素
  • 结束标签：如 </p>

<p class="intro" id="para1">这是一个段落</p>
 ↑       ↑          ↑      ↑        ↑
标签   属性名    属性值   内容    结束标签

二、文本格式化元素

标题 h1 ~ h6

从大到小，h1 最重要

<h1>一级标题</h1>
<h2>二级标题</h2>
<h3>三级标题</h3>

段落与换行

<p> 段落，<br> 换行，<hr> 分割线

<p>段落文本</p>
<p>第一行<br>第二行</p>
<hr>

文本语义标签

标签含义效果
<strong>重要性强调粗体（语义）
<b>视觉加粗粗体（无语义）
<em>重音强调斜体（语义）
<i>视觉倾斜斜体（无语义）
<mark>高亮标记黄色背景
<code>代码片段等宽字体
<del>删除内容删除线
<sub>下标H2O
<sup>上标mc2

三、列表元素

<ul> 无序列表

  • 项目符号（圆点/方块）
  • 无先后顺序
<ul>
  <li>项目 A</li>
  <li>项目 B</li>
</ul>

<ol> 有序列表

  1. 自动编号
  2. 支持 type / start
<ol type="A" start="1">
  <li>项目 A</li>
  <li>项目 B</li>
</ol>

<dl> 定义列表：<dt> 术语 + <dd> 描述

四、链接元素 <a>

超链接是互联网的核心


href 值说明示例
URL外部链接href="https://mdn.com"
相对路径项目内链接href="about.html"
#id锚点跳转href="#section1"
mailto:邮件链接href="mailto:a@b.com"
tel:电话链接href="tel:1380000"

<a href="https://mdn.com" target="_blank" rel="noopener">
  在新标签页打开，且安全
</a>

五、表格元素

<table> 包含：<caption> 标题、<thead> 表头、<tbody> 表体、<tfoot> 表尾


<table>
  <caption>成绩表</caption>
  <thead>
    <tr><th>姓名</th><th>分数</th></tr>
  </thead>
  <tbody>
    <tr><td>张三</td><td>95</td></tr>
  </tbody>
</table>

合并单元格：

  • colspan="2" - 横向合并 2 列
  • rowspan="2" - 纵向合并 2 行

八、语义化结构

语义化标签让页面结构清晰，有利于 SEO 和可访问性


标签语义典型用途
<header>头部页面/区块的顶部
<nav>导航主导航菜单
<main>主内容页面核心内容（唯一）
<article>独立内容文章、帖子、卡片
<section>逻辑分区章节、功能区
<aside>辅助内容侧边栏、引用
<footer>底部版权、底部链接
<details>可折叠FAQ、手风琴
<dialog>对话框模态弹窗

九、全局属性

全局属性可应用于任何 HTML 元素


标识与样式

  • id - 唯一标识
  • class - CSS 类名（可多个）
  • style - 内联样式
  • title - 悬停提示
  • hidden - 隐藏元素

行为控制

  • data-* - 自定义数据
  • tabindex - Tab 导航顺序
  • lang - 内容语言
  • dir - 文字方向
  • draggable - 可拖拽
  • contenteditable - 可编辑
  • spellcheck - 拼写检查

十、事件处理器属性

注意：内联事件处理器用于教学，实际项目推荐 addEventListener()
属性触发时机示例
onclick点击<button onclick="fn()">
onmouseover鼠标悬停鼠标移入触发
oninput输入实时变化输入框实时响应
onchange值改变（失焦后）下拉框选择

onsubmit表单提交表单验证

十一、HTML 最佳实践

DO 推荐做法

  • 使用语义化标签替代 div
  • 始终写 alt 属性
  • 用 <label> 关联表单控件
  • 确保页面有且仅有一个 <h1>
  • 标题层级 h1→h6 递进，不跳级
  • 使用 <meta charset="UTF-8">
  • 添加 viewport 元标签
  • 确保可访问性（键盘导航、屏幕阅读器）

DON'T 避免做法

  • 不要用 table 做布局
  • 不要使用已废弃标签（font、center 等）
  • 不要滥用 <br> 做间距
  • 不要在语义化元素中用空标签
  • 不要内联写大量样式和事件
  • 不要跳过 <html> <head> <body>
  • 不要嵌套块级元素在内联元素中
  • 不要忽略关闭标签

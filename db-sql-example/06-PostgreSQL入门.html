06 - PostgreSQL 入门

世界上最先进的开源关系数据库

PostgreSQL 是一款功能强大的开源对象关系数据库系统，以其稳定性、标准兼容性和扩展性著称。

PostgreSQL 的特点

  • 完全遵守 SQL 标准，支持事务、外键、视图、序列
  • 丰富的扩展：PostGIS（地理空间）、pgvector（向量搜索）、TimescaleDB（时序）
  • 自定义数据类型、函数、操作符
  • MVCC（多版本并发控制），读写互不阻塞
  • 流复制实现高可用，支持逻辑复制

安装 PostgreSQL

macOS、Ubuntu、Windows 安装方式

# macOS
brew install postgresql@16
brew services start postgresql@16

# Ubuntu / Debian
sudo apt update
sudo apt install postgresql postgresql-contrib
sudo systemctl start postgresql

# Windows — 从 https://www.postgresql.org/download/ 下载安装包

# Docker（快速体验）
docker run --name pg -e POSTGRES_PASSWORD=mysecret -d -p 5432:5432 postgres:16

基本管理命令

使用 psql 命令行工具

# 连接数据库
psql -U postgres -h localhost

# 常用元命令
\l                    # 列出所有数据库
\c dbname             # 切换到指定数据库
\d                    # 列出当前库所有表
\d table_name         # 查看表结构
\du                   # 列出所有用户
\df                   # 列出所有函数
\q                    # 退出 psql
-- 创建数据库和用户
CREATE DATABASE myapp;
CREATE USER app_user WITH PASSWORD 'securepass';
GRANT ALL PRIVILEGES ON DATABASE myapp TO app_user;

-- PostgreSQL 特有的数据类型
CREATE TABLE example (
    id       UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    tags     TEXT[],                    -- 数组类型
    metadata JSONB,                    -- JSON 二进制
    location POINT,                    -- 地理坐标
    duration INTERVAL,                 -- 时间间隔
    created_at TIMESTAMPTZ DEFAULT NOW()  -- 带时区时间戳
);

PostgreSQL 特有功能

JSONB 操作

-- 创建含 JSONB 的表
CREATE TABLE articles (
    id      SERIAL PRIMARY KEY,
    title   TEXT,
    content JSONB
);

-- JSONB 查询
INSERT INTO articles (title, content) VALUES (
    'PostgreSQL 入门',
    '{"author": "张三", "tags": ["数据库", "SQL"], "views": 1200}'
);

-- JSONB 操作符
SELECT content->>'author' FROM articles WHERE id = 1;
SELECT * FROM articles WHERE content @> '{"tags": ["数据库"]}';
SELECT jsonb_array_elements_text(content->'tags') FROM articles;

窗口函数

-- 不改变行数的情况下进行聚合计算
SELECT
    employee_name,
    department,
    salary,
    RANK() OVER (PARTITION BY department ORDER BY salary DESC) AS 部门内排名,
    AVG(salary) OVER (PARTITION BY department) AS 部门平均工资
FROM employees;

CTE (公用表表达式)

-- WITH 子句简化复杂查询
WITH regional_sales AS (
    SELECT region, SUM(amount) AS total_sales
    FROM orders GROUP BY region
)
SELECT region, total_sales
FROM regional_sales
WHERE total_sales > 10000;

索引

加速查询的数据库对象

-- B-tree 索引（默认，适合等值和范围查询）
CREATE INDEX idx_orders_amount ON orders(amount);

-- 唯一索引
CREATE UNIQUE INDEX idx_user_email ON users(email);

-- 部分索引（只索引部分数据）
CREATE INDEX idx_active_orders ON orders(status)
WHERE status = 'pending';

-- 表达式索引
CREATE INDEX idx_lower_email ON users(LOWER(email));

-- 查看查询计划
EXPLAIN ANALYZE SELECT * FROM orders WHERE amount > 1000;

备份与恢复

PostgreSQL 的备份工具

# 备份单个数据库
pg_dump -U postgres mydb > mydb_backup.sql

# 备份所有数据库
pg_dumpall -U postgres > all_backup.sql

# 恢复
psql -U postgres mydb < mydb_backup.sql

# 并行备份（大数据量）
pg_dump -U postgres -j 4 mydb > mydb_backup.sql

小结

  • PostgreSQL 是功能最丰富的开源关系数据库
  • 支持 JSONB、数组、全文搜索、地理空间等高级特性
  • psql 命令行工具提供丰富的管理命令
  • 窗口函数和 CTE 是编写复杂查询的利器
  • 索引、备份、用户管理是数据库运维的基础
